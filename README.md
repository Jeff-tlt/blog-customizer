# Blog Customizer

## Описание проекта

Небольшое приложение для настройки внешнего вида статьи.
Параметры меняются через боковую панель и применяются к контенту.

---

## Ссылка на репозиторий

https://github.com/Jeff-tlt/blog-customizer

---

## Функциональность

- Открытие / закрытие панели настроек
- Выбор шрифта
- Изменение размера текста
- Изменение цвета текста
- Изменение фона страницы
- Изменение ширины контента
- Кнопка «Применить»
- Кнопка «Сбросить»

---

## Как запустить

```bash
npm install
npm start
```

Открыть в браузере:
http://localhost:8080

---

## Как устроено

Основное состояние хранится в `App.tsx`.

Форма работает через локальный state (`formState`), изменения применяются только после нажатия на кнопку «Применить».

Стили управляются через CSS-переменные:

- `--font-family`
- `--font-size`
- `--font-color`
- `--bg-color`
- `--container-width`
